Files meant to be used along World-Map-XXX script. See 'AutoHotkey' presets for tagging automation.

'worldMap.json' is pre-built artist-country database provided as starting point or for those without Internet connection. 

It should be copied to '[FOOBAR_PROFILE]\js_data\'.
Where [FOOBAR_PROFILE] should be the current profile folder of the foobar installation.
For ex: 'Z:\Users\xxx\AppData\Roaming\foobar2000\js_data\'
